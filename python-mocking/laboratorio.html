Laboratorio: FastAPI + mock de servicio

Práctica guiada: construir un endpoint que llama a un servicio externo y testearlo sin red.

Objetivo

Construir un endpoint GET /rates/{base} que llama a un servicio externo de tipos de cambio y devuelve un JSON normalizado. En tests, se simula la respuesta del servicio externo sin abrir sockets.

Regla del laboratorio: la app no crea el cliente HTTP dentro del handler. El handler recibe una dependencia (DI) que se puede sobreescribir en tests.

Paso 0 — Estructura mínima

Estructura sugerida (adáptala al proyecto real):

text — estructura
app/
  main.py
  deps.py
  external_fx.py
  routes.py
tests/
  test_rates.py

Paso 1 — Dependencia inyectable (cliente HTTP)

En producción, se crea el cliente real. En tests, se sustituye por otro con transporte simulado.

python — app/deps.py
import httpx

def get_fx_client() -> httpx.AsyncClient:
    # En un proyecto real: cerrar el client con lifespan (startup/shutdown).
    return httpx.AsyncClient(base_url="https://fx.example.com", timeout=5.0)

Paso 2 — Adapter que usa el cliente

Este módulo es el límite de IO: aquí vive la llamada externa. El resto del código sólo ve datos.

python — app/external_fx.py
import httpx

async def fetch_rates(client: httpx.AsyncClient, base: str) -> dict:
    resp = await client.get("/latest", params={"base": base})
    resp.raise_for_status()
    data = resp.json()

    # Normalización mínima para que el resto de la app no dependa del proveedor.
    return {
        "base": data.get("base", base),
        "rates": data.get("rates", {}),
    }

Paso 3 — Endpoint FastAPI (Depends)

El handler pide el cliente a través de Depends. Así se puede sustituir en tests con dependency_overrides.

python — app/routes.py
from fastapi import APIRouter, Depends
import httpx

from .deps import get_fx_client
from .external_fx import fetch_rates

router = APIRouter()

@router.get("/rates/{base}")
async def rates(base: str, client: httpx.AsyncClient = Depends(get_fx_client)):
    return await fetch_rates(client, base=base)
python — app/main.py
from fastapi import FastAPI

from .routes import router

app = FastAPI()
app.include_router(router)

Paso 4 — Test sin red (MockTransport + override)

Se crea un httpx.MockTransport que responde como lo haría el servicio externo. Luego se inyecta un httpx.AsyncClient de test en la dependencia get_fx_client.

python + pytest — tests/test_rates.py
import httpx
import pytest

from app.main import app
from app.deps import get_fx_client


def mock_fx_transport() -> httpx.MockTransport:
    def handler(request: httpx.Request) -> httpx.Response:
        assert request.url.path == "/latest"
        assert request.url.params.get("base") == "EUR"
        return httpx.Response(200, json={"base": "EUR", "rates": {"USD": 1.1}})

    return httpx.MockTransport(handler)


@pytest.mark.asyncio
async def test_rates_uses_mocked_external_service():
    transport = mock_fx_transport()
    fx_client = httpx.AsyncClient(transport=transport, base_url="https://fx.example.com")

    def override_fx_client():
        return fx_client

    app.dependency_overrides[get_fx_client] = override_fx_client

    try:
        asgi = httpx.ASGITransport(app=app)
        async with httpx.AsyncClient(transport=asgi, base_url="http://test") as ac:
            resp = await ac.get("/rates/EUR")

        assert resp.status_code == 200
        assert resp.json() == {"base": "EUR", "rates": {"USD": 1.1}}
    finally:
        app.dependency_overrides.clear()
        await fx_client.aclose()
Punto crítico: el base_url del fx_client de test debe ser el mismo que espera el adapter. Si no coincide, el path puede no resolver como esperas.

Paso 5 — Qué se ha validado

  • El endpoint usa la dependencia (no crea un client interno).
  • La llamada externa sale por un transporte simulado (no hay red real).
  • El handler valida forma de request (path + query params) y devuelve JSON esperado.
