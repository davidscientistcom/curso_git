¿Por qué este curso?

El control de versiones es la herramienta más importante de un desarrollador

Entiende el "Por qué"

No solo te enseñamos comandos. Te explicamos cómo Git guarda los datos (blobs, trees, commits). Cuando entiendes el modelo mental, Git deja de parecer magia negra.

Resolución de Conflictos

El terror de muchos. Aprenderás exactamente por qué ocurren los conflictos, cómo leer las marcas de conflicto (<<<<<<<) y cómo resolverlos con confianza.

Reescribir la Historia

Aprende a usar rebase, commit --amend y el rebase interactivo para mantener un historial limpio y lineal antes de compartir tu código con el equipo.

Filosofía del curso: En Git, casi todo el trabajo es local. Aprenderás a dominar tu repositorio local antes de interactuar con repositorios remotos como GitHub o GitLab, lo que hace que la colaboración sea mucho más sencilla.

El Flujo de Trabajo en Git

Los tres estados fundamentales de un archivo en Git

Working Directory Tus archivos modificados

Módulos del Curso

Navega a través de los diferentes conceptos y ejercicios prácticos

0. SSH y GitHub

Configura tus llaves SSH para conectar de forma segura y sin contraseñas en Windows, Mac y Linux.

ssh-keygen github

1. Fundamentos e Internals

Instalación, configuración inicial, y cómo Git almacena realmente la información (objetos, blobs, trees, commits).

Config Internals

2. Operaciones Básicas

Crear repositorios, el ciclo add-commit, ver historial de cambios, Conventional Commits y cómo ignorar archivos.

add commit log

3. Ramas y Merging

Trabajar en paralelo sin romper nada. Estrategias de Merge (Fast-forward vs 3-way), resolución de conflictos clásicos.

branch merge conflictos

4. Rebase y Reescribir Historia

Mantener un historial limpio. Diferencias entre Merge y Rebase, Rebase Interactivo para limpiar commits.

rebase squash amend

5. Repositorios Remotos

Trabajar con GitHub/GitLab. Clonar, sincronizar, diferencias entre fetch y pull, Pull Requests y stashing temporal.

push pull stash

6. Git Avanzado

Técnicas ninja: git bisect para encontrar bugs, reflog de supervivencia, hooks, submódulos y worktrees.

bisect reflog hooks
